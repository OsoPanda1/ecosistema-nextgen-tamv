# Cognición TAMV - Capa L4

## Capa Cognitiva–Algorítmica
Límites y deberes de las máquinas, IA como instrumento jurídico.

## Componentes Críticos

- **Límites Algorítmicos**: Restricciones éticas en sistemas automatizados
- **IA Jurídica**: Inteligencia artificial como herramienta de justicia
- **Auditoría de Sesgos**: Verificación continua de equidad
- **Transparencia Algorítmica**: Explicabilidad obligatoria
- **Soberanía Cognitiva**: Protección contra manipulación mental

```
├── bias-audit/           # Auditoría de sesgos

```

## Principio Rector

Las máquinas sirven a la dignidad humana, nunca la subordinan.
