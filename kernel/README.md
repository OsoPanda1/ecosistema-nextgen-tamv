# Kernel TAMV - Capas L0–L1

## Capa L0: Ontológica
Define qué existe y qué jamás puede existir en el territorio TAMV.

## Capa L1: Constitucional  
Derechos, deberes, soberanía y mecanismos de enmienda.

## Estructura

```
kernel/
├── ontology/           # Definiciones fundamentales
├── constitution/       # Marco constitucional
├── axioms/            # Axiomas no negociables
├── rights/            # Carta de derechos digitales
└── ethics/            # Ética computacional vinculante
```

## Principio Rector

Estas capas son inmutables por diseño y solo pueden ser modificadas mediante procesos constitucionales específicos que requieren consenso civilizatorio.
